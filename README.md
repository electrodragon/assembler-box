## Assembler Box

`Assembler box` is a toolkit used for assembly language in `DOS BOX`.

### SDK INCLUDES
1. `nasm.exe` (Netwide Assembler)
2. `AFD.EXE` (Advance Fullscreen Debugger)
3. `ALINK.EXE` (LINKER)
4. `MASM.EXE` (Microsoft Macro Assembler)
5. `ndisasm.exe` (Netwide Disassembler)
6. `TASM.EXE` (Turbo Assembler)
7. `TD.EXE` (Turbo Debugger)
8. `RSX.EXE` (RSX-11M operating system)

### Installation

Execute the following command in terminal

```sh
git clone 'https://github.com/electrodragon/assembler-box.git'
```

Mount the `assembler-box/bin` directory in dos box, and it's done.

```text
MOUNT [Drive-Letter] [Local-Directory]
```

To Unmount a Drive

```shell
MOUNT -u [Drive-Letter]
```
